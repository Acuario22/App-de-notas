# App de Notas

Saludos.

Esta es una app de notas sencilla desarrollada con React Native y Expo. Permite agregar tareas, cambiar el tema de la aplicación y organizar notas de forma rápida y simple. La interfaz tiene un diseño limpio y fácil de usar.

**Cómo Ejecutar el Proyecto Localmente**

**Prerrequisitos**
- Tener instalado Node.js.
- Tener instalada la app móvil de Expo Go en tu celular.

**Paso 1: Clonar el repositorio**

Abre tu terminal y clona este proyecto:
```
git clone https://github.com/Acuario22/App-de-notas
```
```
cd app-de-notas
```

**Paso 2: Instalar dependencias**

Instala los paquetes necesarios del proyecto:
```
npm install
```

**Paso 3: Iniciar el servidor de desarrollo de Expo**

Ejecuta el comando para encender el entorno de Expo:
```
npx expo start
```

**Paso 4: Escanear QR y Probar**
- **Dispositivo movil:** Abre la aplicación Expo Go, escanea el código QR que se generará en la terminal y ¡listo! La app se cargará directamente en tu dispositivo.
- **Simulador:** Si tienes configurado un emulador de Android o iOS en tu computadora, presiona a o i en la terminal respectivamente.
